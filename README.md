# Geopolis Consultores

Sitio web corporativo de **Sociedad de Profesionales Geopolis Consultores Limitada** — consultoría geoespacial, estudios ambientales e instrumentos de planificación territorial en Chile.

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla (sin frameworks)
- Google Fonts: Space Grotesk · Inter · JetBrains Mono
- Font Awesome 6.4.0
- Canvas API (animación de partículas y mockup geoespacial)
- IntersectionObserver (scroll reveal + contadores animados)
